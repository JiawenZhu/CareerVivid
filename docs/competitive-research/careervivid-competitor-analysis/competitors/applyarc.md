# ApplyArc — Competitor Profile

> **Threat Level: 🟡 MEDIUM**
> **Category fit:** Entrepreneurship & Job Creation
> **XPRIZE risk:** 17-tool platform, £19/mo, growing in UK market — may enter XPRIZE

---

## Overview
ApplyArc is a UK-based AI career platform offering 17 career tools in one subscription. Targeting the UK and European markets, they've built a broad feature set at a competitive £19/month price point. Less known in the US market but growing, and their breadth of tools makes them a credible XPRIZE entrant if they can demonstrate revenue.

## Key Capabilities
| Feature | Detail |
|---|---|
| AI Resume Builder | GPT-powered, ATS optimized |
| Cover Letter Generator | Per-job-description generation |
| Interview Simulator | Text-based mock interviews |
| LinkedIn Message Crafter | AI cold outreach messages |
| Salary Negotiation Coach | AI negotiation scripts |
| Job Description Analyzer | Keyword and requirement extraction |
| Thank You Note Writer | Post-interview follow-ups |
| Reference Request Writer | AI-drafted reference request emails |
| Application Email Writer | Cold application emails |
| Career Change Advisor | AI career path suggestions |
| *+ 7 more tools* | All GPT-based |

## Business Model & Revenue
| Tier | Price | Features |
|---|---|---|
| Free | £0 | 3 tool uses/mo |
| Pro | £19/mo | Unlimited, all 17 tools |

**Revenue estimate:** Unknown user count; UK market suggests smaller absolute scale than US competitors.

## Tech Stack
- **AI:** GPT-4 (OpenAI) — **NOT Gemini**, XPRIZE weakness
- **Stack:** Likely Next.js + Supabase (common UK SaaS stack)
- **No Chrome extension, no CLI, no MCP server**

## Weaknesses vs CareerVivid
- GPT-based, not Gemini
- UK-centric — limited US distribution
- No autonomous apply
- No voice interview coaching
- No job tracker / Kanban
- No B2B employer portal
- No CLI or MCP server
- No Google Cloud footprint
- Breadth without depth — 17 tools but none as sophisticated as CareerVivid's individual features

## XPRIZE Assessment
ApplyArc could enter XPRIZE but their UK focus and GPT dependency limit their scoring potential. CareerVivid's US market scale and Gemini stack outcompetes on all three judging criteria.

## What to Steal
- The "17 tools in one" positioning is compelling marketing even if the tools are simple — CareerVivid should count its distinct AI features and market them: "8 AI-powered career tools, one platform"
- Salary negotiation coaching is a feature gap in CareerVivid — high perceived value, relatively easy to implement as a Gemini chat mode
