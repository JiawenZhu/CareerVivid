# CareerVivid CLI · Documentation

Global overview of the `cv` command-line interface.

## Quick Installation

```bash
npm install -g careervivid
```

## Command Index

- [`cv publish`](./publish.md) — Share articles and diagrams.
- [`cv whiteboard`](./whiteboard.md) — Manage Mermaid architecture diagrams.
- [`cv auth`](./auth.md) — Account and API key management.
- [`cv config`](./config.md) — Local settings and environment tuning.

## General Commands

- `cv -v` — Show currently installed version.
- `cv update` — Update the CLI to the latest version.
- `cv --help` — Show global help or help for a specific command.
