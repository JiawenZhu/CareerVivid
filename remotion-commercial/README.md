# CareerVivid Remotion Commercial

Standalone Remotion source for a CareerVivid commercial grounded in the current product:

- AI resume builder
- Interview Studio
- Portfolio publishing
- Job tracker
- Browser extension workflow

## Commands

```bash
cd remotion-commercial
npm install
npm run preview
npm run render
```

The default render target is:

`renders/careervivid-commercial.mp4`
