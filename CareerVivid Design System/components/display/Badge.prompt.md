Badge (status chip) + StatusDot (pipeline dot). Status color lives ONLY in these small elements — never large blocks.

```jsx
<Badge tone="success">Offered</Badge>
<Badge tone="warning" dot>High priority</Badge>
<StatusDot status="Interviewing" />
```

Tones: accent, success, warning, danger, info, neutral. 11px/700, 2×8px padding, pill.
