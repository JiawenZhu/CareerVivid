Lucide icon wrapper — use for every glyph in CareerVivid UI (the app uses lucide-react v0.344.0).

```jsx
<Icon name="briefcase" size={15} />
<Icon name="wand-2" size={16} style={{ color: '#625bd5' }} />
```

Available names: briefcase, file-text, mic, search, sparkles, wand-2, check-circle-2, alert-circle, calendar-clock, arrow-up-right, arrow-right, layout-dashboard, message-square-text, chevron-left/right/down, x, plus, external-link, clipboard-list, target, filter, user-round, map-pin, dollar-sign, settings, bell, trophy, send, clock-3, moon, sun, bot, refresh-cw, shield-check, book-open, presentation, palette, trash-2, download, share-2, link, eye, pen-line, globe, loader-2, play, square, home, kanban-square, app-window, more-horizontal, log-out, graduation-cap, building-2, zap, star, menu … (see components/icons/lucideIconData.js).

Sizes: 11px inside chips, 13–15px in compact rows, 16–18px in buttons and icon wells.
