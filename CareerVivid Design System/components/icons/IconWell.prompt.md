Tinted icon well — the standard CareerVivid icon container (28–40px square, tinted bg + matching icon color). Use for stat cards, feature rows, list-item leads.

```jsx
<IconWell icon="briefcase" tone="purple" size={32} />
<IconWell icon="mic" tone="rose" size={40} />
```

Tones: purple (default), softPurple, amber, green, rose, blue, neutral. Prefer tinted wells over saturated panels.
